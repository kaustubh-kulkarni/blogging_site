Blogging site

This is a project for creating a personal blogging site where a person can upload personal blogs.

This will be CRUD application and the data/blogs that we are adding will be stored in database.

User model - 
Name, password, email

Blog-
Id, content

User will have a list of blogs as one user can create multiple blogs.

The ID we will be using would be GUIDS



Project tech - 
WEBAPI using c#
Client app using angular
POSTGERS DB
